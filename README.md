# 🚀 KodeKloud Level 1 - Kubernetes Practice

This repository contains hands-on labs and exercises from **KodeKloud Level 1 Kubernetes**. It focuses on building a strong foundation in Kubernetes through practical, real-world scenarios.

---

## 📌 Overview

The tasks in this repository cover essential Kubernetes concepts such as deploying applications, managing workloads, configuring resources, and troubleshooting issues.

Each lab is designed to simulate real DevOps challenges and improve practical skills using `kubectl` in a live cluster environment.

---

## 🎯 What You Will Learn

### 🧱 Core Kubernetes Concepts
- Understanding Pods and how to deploy them
- Managing applications using Deployments and ReplicaSets
- Working with Kubernetes Namespaces

### ⚙️ Application Management
- Performing rolling updates and rollbacks
- Scaling and updating applications
- Managing application lifecycle in a cluster

### 📊 Resource Management
- Setting CPU and memory limits
- Optimizing resource usage for containers

### 🔁 Workloads & Scheduling
- Running batch jobs and CronJobs
- Scheduling and automating workloads

### 🌐 Networking & Services
- Exposing applications using Services
- Understanding NodePort and cluster networking basics

### 🛠 Troubleshooting & Debugging
- Identifying and fixing deployment issues
- Debugging misconfigured Pods and workloads

---

## 🛠 Tools & Technologies

- Kubernetes
- kubectl (CLI tool)
- YAML configurations
- Containerized applications
